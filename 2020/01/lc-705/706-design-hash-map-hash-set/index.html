Design HashSet

Link: Design HashSet

Problem description

Design a HashSet without using any built-in hash table libraries.

To be specific, your design should include these functions:

  • add(value): Insert a value into the HashSet.
  • contains(value) : Return whether the value exists in the HashSet or not.
  • remove(value): Remove a value in the HashSet. If the value does not exist in the HashSet, do nothing.

Example:

MyHashSet hashSet = new MyHashSet();
hashSet.add(1);         
hashSet.add(2);         
hashSet.contains(1);    // returns true
hashSet.contains(3);    // returns false (not found)
hashSet.add(2);          
hashSet.contains(2);    // returns true
hashSet.remove(2);          
hashSet.contains(2);    // returns false (already removed)

Note:

  • All values will be in the range of [0, 1000000].
  • The number of operations will be in the range of [1, 10000].
  • Please do not use the built-in HashSet library.

Method

Use a linked list array to obtain O(1) get, and remove. Use linked list to handle hash collision.

Code

public class MyHashSet_705 {
    LinkedList<Integer>[] arr;
    int MAX_SIZE = 256;
    int size;

    /**
     * Initialize hash set.
     */
    public MyHashSet() {
        arr = new LinkedList[MAX_SIZE];
        size = 0;
    }

    /**
     * Add value to hash set.
     * Note that hash set will not contains duplicated value.
     *
     * @param key value to be added
     */
    public void add(int key) {
        int index = key % MAX_SIZE;

        if (arr[index] == null) {
            arr[index] = new LinkedList<>();
        }

        LinkedList<Integer> tmp = arr[index];

        for (int i : tmp) {
            if (i == key) {
                return;
            }
        }
        tmp.add(key);

        if (size >= MAX_SIZE * 0.75) {
            rehashing();
        }
    }

    /**
     * Remove value from hash set.
     *
     * @param key value to be removed from hash set.
     */
    public void remove(int key) {
        int index = key % MAX_SIZE;
        if (arr[index] == null) {
            return;
        }
        for (int i = 0; i < arr[index].size(); i++) {
            if (arr[index].get(i).equals(key)) {
                arr[index].remove(i);
                size--;
                return;
            }
        }
    }

    /**
     * Rehashing to make hash set more balanced.
     */
    private void rehashing() {
        MAX_SIZE *= 2;
        LinkedList<Integer>[] newArr = new LinkedList[MAX_SIZE];

        for (List<Integer> l : arr) {
            for (int i : l) {
                int index = i % MAX_SIZE;
                if (newArr[index] == null) {
                    newArr[index] = new LinkedList<>();
                }
                newArr[index].add(i);
            }
        }
        arr = newArr;
    }

    /**
     * Returns true if this set contains the specified element
     */
    public boolean contains(int key) {
        int index = key % MAX_SIZE;

        if (arr[index] == null) {
            return false;
        }
        for (int i : arr[index]) {
            if (i == key) {
                return true;
            }
        }
        return false;
    }
}

Design HashMap

Link:

Problem description

Design a HashMap without using any built-in hash table libraries.

To be specific, your design should include these functions:

  • put(key, value) : Insert a (key, value) pair into the HashMap. If the value already exists in the HashMap, update the value.
  • get(key): Returns the value to which the specified key is mapped, or -1 if this map contains no mapping for the key.
  • remove(key) : Remove the mapping for the value key if this map contains the mapping for the key.

Example:

MyHashMap hashMap = new MyHashMap();
hashMap.put(1, 1);          
hashMap.put(2, 2);         
hashMap.get(1);            // returns 1
hashMap.get(3);            // returns -1 (not found)
hashMap.put(2, 1);          // update the existing value
hashMap.get(2);            // returns 1 
hashMap.remove(2);          // remove the mapping for 2
hashMap.get(2);            // returns -1 (not found)

Note:

  • All keys and values will be in the range of [0, 1000000].
  • The number of operations will be in the range of [1, 10000].
  • Please do not use the built-in HashMap library.

Method

Same as previous question, but use a

Code

public class MyHashMap {
    private LinkedList<Cell>[] map;
    int MAX_SIZE = 256;
    int size;

    /**
     * Initialize map.
     */
    public MyHashMap() {
        this.map = new LinkedList[MAX_SIZE];
    }

    /**
     * Value will always be non-negative.
     *
     * @param key   given key
     * @param value given value
     */
    public void put(int key, int value) {
        int index = key % MAX_SIZE;
        if (map[index] == null) {
            map[index] = new LinkedList<>();
        }

        Cell tmp = new Cell(key, value);
        LinkedList<Cell> l = map[index];

        for (Cell c : l) {
            if (c.key == key) {
                l.remove(c);
                break;
            }
        }

        l.add(tmp);
        size++;

        if (size >= MAX_SIZE * 0.75) {
            rehashing();
        }
    }

    /**
     * Returns the value to which the specified key is mapped, or -1 if this map contains no mapping for the key
     *
     * @param key given key
     * @return corresponding value
     */
    public int get(int key) {
        int index = key % MAX_SIZE;
        if (map[index] == null) {
            return -1;
        }

        LinkedList<Cell> l = map[index];

        for (Cell c : l) {
            if (c.key == key) {
                return c.val;
            }
        }

        return -1;
    }

    /**
     * Removes the mapping of the specified value key if this map contains a mapping for the key
     *
     * @param key given key
     */
    public void remove(int key) {
        int index = key % MAX_SIZE;
        if (map[index] == null) {
            return;
        }

        LinkedList<Cell> l = map[index];

        l.removeIf(c -> c.key == key);

        size--;
    }

    /**
     * Rehashing to make hash set more balanced.
     */
    private void rehashing() {
        MAX_SIZE *= 2;

        LinkedList<Cell>[] tmp = new LinkedList[MAX_SIZE];

        for (List<Cell> l : map) {
            if (l != null) {
                for (Cell c : l) {
                    int index = c.key % MAX_SIZE;
                    if (tmp[index] == null) {
                        tmp[index] = new LinkedList<>();
                    }
                    tmp[index].add(c);
                }
            }
        }

        map = tmp;
    }

    /**
     * Cell used in map.
     */
    static class Cell {
        int key;
        int val;

        public Cell(int key, int val) {
            this.key = key;
            this.val = val;
        }
    }
}

Follow Up: Design Generic HashMap/HashSet

HashSet is actually based on HashMap, therefore, implement hash map first.

Cell used in HashMap

/**
 * Design a cell for hash map.
 */
public class Cell<K, V> {
    private K key;
    private V value;

    /**
     * Constructor of cell.
     *
     * @param key given key of current cell
     * @param val given value of current cell
     */
    public Cell(K key, V val) {
        this.key = key;
        this.value = val;
    }

    /**
     * Return the hash code of key.
     * If key is null, return 0.
     * Otherwise, return the normal hash code.
     *
     * @return hash code of key.
     */
    @Override
    public int hashCode() {
        return this.key == null ? 0 : this.key.hashCode();
    }

    /**
     * Check if given object equals to this object.
     *
     * @param o given object
     * @return true if given object equals to this object.
     */
    @Override
    public boolean equals(Object o) {
        if (o instanceof Cell<?, ?>) {
            Cell<K, V> that = (Cell<K, V>) o;       // cast object if available

            if (this.key == null) {
                return that.key == null;
            } else {
                return this.key.equals(that.key);
            }

        } else {
            return false;
        }
    }

    /**
     * Compare if given key equals the current cell's key.
     *
     * @param key compare key
     * @return true if given key equals the current cell's key
     */
    public boolean keyEquals(K key) {
        if (this.key == null) {
            return key == null;
        } else {
            return this.key.equals(key);
        }
    }

    /**
     * Return key of current cell.
     *
     * @return key of current cell
     */
    public K getKey() {
        return key;
    }

    /**
     * Return value of current cell.
     *
     * @return value of current cell
     */
    public V getValue() {
        return value;
    }

    /**
     * Convert cell to string.
     *
     * @return key and value in cell
     */
    @Override
    public final String toString() {
        return key.toString() + "=" + value.toString();
    }
}

Implementation of HashMap

import java.util.*;

/**
 * Implementation of hash map.
 */

// TODO: Multi map

public class DesignHashMap<K, V> {
    private K key;
    private V value;
    private final static double LOAD_FACTOR = 0.75d;        // load factor of hash map that supposes to be 0.75
    private int MAX_SIZE = 256;                             // max size of hash map, resizeable

    private List<Cell<K, V>>[] items;                       // implement of hash map bucket
    private int size;

    /**
     * Constructor of hash map.
     */
    public DesignHashMap() {
        this.items = (List<Cell<K, V>>[]) new LinkedList[MAX_SIZE];
        size = 0;
    }

    /**
     * Constructor overload with specific capacity.
     *
     * @param capacity given capacity
     */
    public DesignHashMap(int capacity) {
        this.items = (List<Cell<K, V>>[]) new LinkedList[capacity];
        size = 0;
    }

    /**
     * Calculate the hash code of given key.
     *
     * @param key given key
     * @return hash code of key
     */
    public int hashCodeOfKey(K key) {
        return (key == null) ? 0 : Math.abs(key.hashCode() % MAX_SIZE);
    }

    /**
     * Put new cell or update cell (as entry) in hash map.
     *
     * @param key   new cell key
     * @param value new cell value
     * @return old value in hash map entry when updating hash map entry, or null if it is a new entry
     */
    public V put(K key, V value) {
        Cell<K, V> cell = new Cell<>(key, value);
        int index = hashCodeOfKey(key);
        V old = null;

        if (items[index] == null) {
            items[index] = new LinkedList<>();
        }

        List<Cell<K, V>> slot = items[index];

        for (Cell<K, V> c : slot) {
            if (cell.equals(c)) {
                old = c.getValue();
                slot.remove(c);
                this.size--;
                break;
            }
        }

        this.size++;
        slot.add(cell);

        if (this.size > MAX_SIZE * LOAD_FACTOR) {
            rehashing();
        }

        return old;
    }

    /**
     * Get value based on given key
     *
     * @param key given key
     * @return value in hash map based on key
     */
    public V get(K key) {
        int index = hashCodeOfKey(key);

        if (items[index] == null) {
            return null;
        }

        List<Cell<K, V>> slot = items[index];

        for (Cell<K, V> cell : slot) {
            if (cell.keyEquals(key)) {
                return cell.getValue();
            }
        }

        return null;
    }

    /**
     * Remove entry from map by key.
     *
     * @param key given key to remove entry
     * @return if entry is successfully removed, if entry is not existed in map, return false
     */
    public boolean remove(K key) {
        int index = hashCodeOfKey(key);

        if (items[index] == null) {
            return false;
        }

        List<Cell<K, V>> l = items[index];

        for (Cell<K, V> c : l) {
            if (c.keyEquals(key)) {
                l.remove(c);
                size--;
                return true;
            }
        }

        return false;
    }

    /**
     * Check if given key exists in map.
     *
     * @param key given key
     * @return if key exists in map
     */
    public boolean containsKey(K key) {
        int index = hashCodeOfKey(key);

        if (items[index] == null) {
            return false;
        }

        List<Cell<K, V>> l = items[index];

        for (Cell<K, V> c : l) {
            if (c.keyEquals(key)) {
                return true;
            }
        }

        return false;
    }

    /**
     * Double the size of hash map if its size is larger than the limit.
     */
    private void rehashing() {
        MAX_SIZE *= 2;
        List<Cell<K, V>>[] resize = (List<Cell<K, V>>[]) new LinkedList[MAX_SIZE];

        for (List<Cell<K, V>> slot : items) {       // deep copy
            if (slot != null) {
                for (Cell<K, V> cell : slot) {
                    int index = hashCodeOfKey(cell.getKey());       // new index
                    if (resize[index] == null) {
                        resize[index] = new LinkedList<Cell<K, V>>();
                    }
                    resize[index].add(cell);
                }
            }
        }

        this.items = resize;
    }

    /**
     * Convert map to string.
     *
     * @return string contains all entries in map
     */
    @Override
    public String toString() {
        if (this.size == 0) {
            return "{}";
        }

        StringBuilder sb = new StringBuilder();
        sb.append("{");

        for (List<Cell<K, V>> i : this.items) {
            if (i != null) {
                for (Cell<K, V> c : i) {
                    sb.append(c.toString());
                    sb.append(", ");
                }
            }
        }

        sb.delete(sb.length() - 2, sb.length());
        sb.append("}");

        return sb.toString();
    }
}

Implementation of HashSet based on HashMap

/**
 * Implement a hash set.
 * Note that the actual implement method is to use a hash map to implement the hash set.
 * Use null as the value object in hash map.
 */

public class DesignHashSet<E> {
    private DesignHashMap<E, Object> map;
    private static final Object dummy = new Object();       // dummy value to associate with an Object in Map

    /**
     * Constructor of set.
     */
    public DesignHashSet() {
        map = new DesignHashMap<>();
    }

    /**
     * Constructor overload with specific capacity.
     *
     * @param capacity given capacity
     */
    public DesignHashSet(int capacity) {
        map = new DesignHashMap<>(capacity);
    }

    /**
     * Add element into set.
     *
     * @param e element to be added
     * @return if element is successfully added, or return false if element has been in set
     */
    public boolean add(E e) {
        return map.put(e, dummy) == null;
    }

    /**
     * Remove element in set.
     *
     * @param e element to be removed
     * @return if element is successfully remove, or return false if element is not in set
     */
    public boolean remove(E e) {
        return map.remove(e);
    }

    /**
     * Check if given element exists in set.
     *
     * @param e given element
     * @return if element exists in set
     */
    public boolean contains(E e) {
        return map.containsKey(e);
    }
}
