Set up Github Page

April 2, 2019

It is easy to create a GitHub page to host your own blog. Although it does not support backend function, such as comment, it can still use issue that relate to this blog's repository to create "comment".

Also, GitHub pages can host a static web page for demonstration purpose, such as showing a project. It can avoid the cost of server and domain.

Steps:

  1. The most basic step, create a GitHub account and copy your username.
  2. Then create a new repository.
  3. The name of this repository should be your YOUR_USERNAME.io. Create repository.
  4. Add README.md and index.html
  5. This index.html is the entry of GitHub page. Modify this page and add more to your blog!
