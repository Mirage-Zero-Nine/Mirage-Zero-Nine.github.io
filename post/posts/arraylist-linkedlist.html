ArrayList and LinkedList in Java

June 01, 2019

Implement Type

  • ArrayList - Dynamic array.
  • LinkedList - Doubly linked list.

Insertion and removal

  • LinkedList has a better performance. LinkedList has no risk of resizing array copying content to new array if array gets full (double linked list).
  • When inserting, adding into ArrayList takes O(n) in worst case (list is full), O(1) in LinkedList.
  • ArrayList also needs to be update its index if inserted at anywhere except the end of array.
  • Removal in LinkedList is better than ArrayList based on same reason.

Memory

  • LinkedList has more memory overhead than ArrayList.
  • In ArrayList, each index only holds actual object (data).
  • In LinkedList, each node holds both data and address of next and previous node.

Find

  • Both LinkedList and ArrayList require O(n) time to find if an element is present or not.
  • ArrayList can do Binary Search when it is sorted and therefore can search in O(Log n) time.
