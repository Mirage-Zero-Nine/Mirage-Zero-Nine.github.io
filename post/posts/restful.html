RESTful Routes

Table of all 7 RESTful routes

Name Path Verb Purpose Mongoose Method
Index /posts GET List all posts Post.find()
New /posts/new GET Show new post form N/A
Create /posts POST Create a new post, then redirect somewhere Post.create()
Show /posts/:id GET Show info about one specific post Post.findById()
Edit /posts/:id/edit GET Show edit form for one post Post.findById()
Update /posts/:id PUT Update particular posts, then redirect somewhere Post.findByIdAndUpdate()
Destroy /posts/:id DELETE Delete a particular post, then redirect somewhere Post.findByIdAndRemove()
