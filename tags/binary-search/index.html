LC 1011 1482 - Binary Search: Define Search Range

However, sometimes the search target can be a specific range that is differ from given collection. The meaning of binary search is identical to normal problems, that to remove half of the all possible values until the target was found. For example, the search range could be the min and max possible value in given collection.
