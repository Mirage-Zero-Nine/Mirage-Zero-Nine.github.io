Invert Binary Tree

June 11, 2019

Link: Invert Binary Tree

Problem description

Invert a binary tree.

Example:

Input:

     4
   /   \
  2     7
 / \   / \
1   3 6   9

Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1

Method

DFS traversal to the bottom of tree and swap the child element of each root element.

Code

/**
 * Traverse to bottom to tree and invert child of it.
 *
 * @param root root node
 * @return inverted tree
 */
public TreeNode invertTree(TreeNode root) {

    /* End point */
    if (root == null) {
        return root;
    }

    root.left = invertTree(root.left);
    root.right = invertTree(root.right);

    TreeNode tmp = root.left;
    root.left = root.right;
    root.right = tmp;

    return root;
}
