Binary Search Tree Iterator

June 13, 2019

Link: Binary Search Tree Iterator

Problem description

Implement an iterator over a binary search tree (BST). Your iterator will be initialized with the root node of a BST.

Calling next() will return the next smallest number in the BST.

Example:

BSTIterator iterator = new BSTIterator(root);
iterator.next();    // return 3
iterator.next();    // return 7
iterator.hasNext(); // return true
iterator.next();    // return 9
iterator.hasNext(); // return true
iterator.next();    // return 15
iterator.hasNext(); // return true
iterator.next();    // return 20
iterator.hasNext(); // return false

Note:

  • next() and hasNext() should run in average O(1) time and uses O(h) memory, where h is the height of the tree.
  • You may assume that next() call will always be valid, that is, there will be at least a next smallest number in the BST when next() is called.

Method

Split one stack used in in-order traversal into multi parts. Each time only push leftmost elements into stack, so that it could work as “controlled stack”.

First of all, traverse to leftmost node in BST and push all visited elements into stack.

Then when next() is called, pop the top of stack. Check whether this node has right child. If so, repeat find leftmost this process and push visited nodes into stack.

hasNext() can be implemented by checking the size of stack.

Code

public class BSTIterator {

    private Stack<TreeNode> s = new Stack<>();

    /**
     * Implement a In-order traversal with a stack.
     *
     * @param root given root node
     */
    public BSTIterator(TreeNode root) {
        this.leftMost(root);
//        inorder(root);
    }

    /**
     * @return the next smallest number
     */
    public int next() {
        TreeNode next = s.pop();
        if (next.right != null) {
            leftMost(next.right);
        }
        return next.val;
    }

    /**
     * @return whether we have a next smallest number
     */
    public boolean hasNext() {
        return !this.s.isEmpty();
    }

    private void leftMost(TreeNode r) {
        while (r != null) {
            this.s.push(r);
            r = r.left;
        }
    }
}
