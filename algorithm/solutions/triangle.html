Triangle

June 09, 2019

Link: Triangle

Problem description

Given a triangle, find the minimum path sum from top to bottom. Each step you may move to adjacent numbers on the row below.

For example, given the following triangle

[
     [2],
    [3,4],
   [6,5,7],
  [4,1,8,3]
]

The minimum path sum from top to bottom is 11 (i.e., 2 + 3 + 5 + 1 = 11).

Note:

Bonus point if you are able to do this using only O(n) extra space, where n is the total number of rows in the triangle.

Method

The path required is from top to bottom, however, the finding process will be more easily if finding from bottom to top. Each time, compare two adjacent numbers on the row below and find the smaller one to add on current value, until reaches the top level.

Code

/**
 * Dynamic programming that find path from bottom to top.
 * Note that only adjacent numbers on the row below is reachable.
 * Therefore, simply compare two adjacent element will suffice.
 *
 * @param triangle given triangle
 * @return min distance
 */
public int minimumTotal(List<List<Integer>> triangle) {

    /* Corner case */
    if (triangle.size() == 0) {
        return 0;
    }

    int[] dp = new int[triangle.get(triangle.size() - 1).size()];       // get last level of triangle

    for (int i = 0; i < triangle.get(triangle.size() - 1).size(); i++) {
        dp[i] = triangle.get(triangle.size() - 1).get(i);       // fill value with last level of triangle
    }

    for (int i = triangle.size() - 2; i > -1; i--) {
        for (int j = 0; j < triangle.get(i).size(); j++) {
            dp[j] = Math.min(dp[j], dp[j + 1]) + triangle.get(i).get(j);
        }
    }

    return dp[0];
}

It fits the requirement that only takes O(n) extra space.
