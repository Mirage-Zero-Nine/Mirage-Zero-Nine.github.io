CSS @media

April 9, 2019
  • Set background-color to lavender when viewport <= 800px
  • Set background-color to lightgreen when 400px <= viewport <= 799px
  • Set background-color to lightblue when viewport <= 399px
body {
    background-color: lightblue;
}

@media screen and (min-width: 400px) {
    body {
        background-color: lightgreen;
    }
}

@media screen and (min-width: 800px) {
    body {
        background-color: lavender;
    }
}
