Install Gem5

April 11, 2019

First, basic dependency

Install following dependency via apt.

  • git: sudo apt-get install git
  • gcc 4.8+: sudo apt-get install build-essential
  • SCons: sudo apt-get install scons
  • Python 2.7 (Gem5 does not support Python 3): sudo apt-get install python-dev
  • m4: sudo apt install m4
  • sudo apt-get install libboost-dev

Second, install protobuf

ATTENTION: protobuf version should be less than 3.5.1. Version above that would cause "all warnings being treated as errors and terminate build gem5 process.

Following instructions below:

  • Download protobuf version 3.5.1 from here.
  • Move to protobuf folder and execute following command:
  • ./configure
  • make
  • make check
  • sudo make install
  • sudo ldconfig # refresh shared library cache.

Building

This following command can be regrad a "Hello World" to test if this Gem5 can work.

scons build/X86/gem5.opt -j2

Note: -j2 is the para that decide how many threads using in this build process. For instance, taking 4 threads can use -j4 as parameter.

This building process can take a very long time, half an hour maybe. Therefore, have something handy to kill the time.
