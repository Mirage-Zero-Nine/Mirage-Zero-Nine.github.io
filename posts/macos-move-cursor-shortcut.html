Move Cursor in macOS

April 29, 2019

Move cursor:

Move the cursor by character, word, line, etc.

  • ^ + F: Moving the cursor to previous character
  • ^ + B: Moving the cursor to next character
  • ⌃ + ⌥ + B: Moving the cursor to previous word
  • ⌃ + ⌥ + F: Moving the cursor to next word
  • ^ + A: Moving the cursor to line begin
  • ^ + E: Moving the cursor to line end
  • ^ + L: Moving the cursor to middle of screen

Select text:

Extend the selection by character, word, paragraph, and line.

  • ^ + ⇧ + B: Select previous character
  • ^ + ⇧ + F: Select next character
  • ⌃ + ⌥ + ⇧ + B: Select previous word
  • ⌃ + ⌥ + ⇧ + F: Select next word
  • ^ + ⇧ + A: Select to line begin
  • ^ + ⇧ + E: Select to line end
  • ^ + ⇧ + P: Select to previous line
  • ^ + ⇧ + N: Select to next line

Delete text:

Remove characters, words, and lines.

  • ^ + H: Delete previous character
  • ^ + D: Delete next character
  • ^ + K: Delete to line end
