Install and update RSSHub via Docker

April 5, 2019

Install RSSHub

  1. Install Docker.
  2. Make sure you have the correct token/key for corresponding service, such as Twitter, Youtube.
  3. Set those keys as environment in Docker command, as listed below. You can add or remove environment variables as your requirement.
  4. Pull and run the container into local machine. You can access it via the specific port you assigned. 
docker run -d --name rsshub --restart=always \
-p 25890:1200 \
-e HTTP_BASIC_AUTH_NAME='YOUR_AUTH_NAME' \
-e HTTP_BASIC_AUTH_PASS='YOUR_AUTH_PASS' \
-e TWITTER_CONSUMER_KEY='YOUR_TWITTER_TOKEN' \
-e TWITTER_CONSUMER_SECRET='YOUR_TWITTER_CONSUMER_SECRET' \
-e TWITTER_ACCESS_TOKEN='YOUR_TWITTER_TOKEN' \
-e TWITTER_ACCESS_TOKEN_SECRET='YOUR_TWITTER_ACCESS_TOKEN_SECRET' \
-e YOUTUBE_KEY='YOUR_YOUTUBE_KEY' \
diygod/rsshub

Update RSSHub

The basic idea of updating a Docker container is to remove the container and re-pull the image and container to local machine. This command includes removing image and container process. 

docker stop rsshub && docker rm rsshub && docker rmi diygod/rsshub && docker run -d --name rsshub --restart=always \
-p 25890:1200 \
-e HTTP_BASIC_AUTH_NAME='YOUR_AUTH_NAME' \
-e HTTP_BASIC_AUTH_PASS='YOUR_AUTH_PASS' \
-e TWITTER_CONSUMER_KEY='YOUR_TWITTER_TOKEN' \
-e TWITTER_CONSUMER_SECRET='YOUR_TWITTER_CONSUMER_SECRET' \
-e TWITTER_ACCESS_TOKEN='YOUR_TWITTER_TOKEN' \
-e TWITTER_ACCESS_TOKEN_SECRET='YOUR_TWITTER_ACCESS_TOKEN_SECRET' \
-e YOUTUBE_KEY='YOUR_YOUTUBE_KEY' \
diygod/rsshub

Environment variable should be your own.
