Debugging iPhone Safari Web Page on macOS

April 6, 2019

Maybe only for fun, but sometimes it is required to open Web Inspector to show web page elements and resources on Safari.

And the most importantly, open web inspector is a best way for debugging if web page is not organized as expected.

Here are the steps:

  1. Go to iPhone Setting
  2. Follow the path: "Safari" -> "Advanced", enable JavaScript and Web Inspector.
  3. Now open your Mac and open Safari.
  4. Click "Safari" on top bar of screen, then open "Preferences".
  5. Open "Advanced" tab, check the box next to “Show Develop menu in menu bar”
  6. Link iOS device to Mac, open the web page to be inspected.
  7. Now on the top bar, find "Develop" tab, click it. Under that tab, iOS device name will show up there if everything works well.
  8. Open the web page title to start inspection.
