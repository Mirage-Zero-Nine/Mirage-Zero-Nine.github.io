LC 168/171 - Base Conversion

October 22, 2019 in Algorithm

Excel Sheet Column Title

Link: Excel Sheet Column Title

Problem description

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:

    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...

Example 1:

Input: 1
Output: "A"

Example 2:

Input: 28
Output: "AB"

Example 3:

Input: 701
Output: "ZY"

Method

Decimal to base-26.

Note that the mapping rule starts at 1, instead of 0 in normal conversion.

Code

/**
 * Decimal to base-26.
 * Note that the mapping rule starts at 1, instead of 0 in normal conversion.
 *
 * @param n given column # in Excel
 * @return corresponding column title as appear in an Excel sheet
 */
public String convertToTitle(int n) {
    if (n == 0) {
        return "";
    }
    StringBuilder sb = new StringBuilder();
    while (n > 0) {
        int tmp = (n - 1) % 26;
        sb.append((char) (tmp + 'A'));
        n = (n - 1) / 26;
    }

    return sb.reverse().toString();
}

Excel Sheet Column Number

Link: Excel Sheet Column Number

Problem description

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:

    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...

Example 1:

Input: "A"
Output: 1

Example 2:

Input: "AB"
Output: 28

Example 3:

Input: "ZY"
Output: 701

Method

Base-26 to decimal.

c1 * 26^(n - 1) + c2 * 26^(n - 2) + ... + cn * 26^(0), where n is the length of given string.

Code

/**
 * Base-26 to decimal.
 * c1 * 26^(n - 1) + c2 * 26^(n - 2) + ... + cn * 26^(0), where n is the length of given string.
 *
 * @param s given base-26 string
 * @return representing decimal number
 */
public int titleToNumber(String s) {
    int n = s.length();
    int out = 0;
    for (int i = 0; i < n; i++) {
        out += ((s.charAt(i) - 'A' + 1) * (int) Math.pow(26, n - i - 1));
    }
    return out;
}
