Oh-my-zsh config

Install on Ubuntu on Amazon EC2

  1. Connect to EC2 instance
  2. Install zsh : sudo apt-get update && sudo apt-get install zsh
  3. Edit passwd configuration file to tell which shell to use for user ubuntu : sudo nano /etc/passwd
  4. Look for ubuntu user, and replace bin/bash by bin/zsh
  5. Install OhMyZsh : sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
  6. Disconnect from your instance and reconnect it to instance.

Install on macOS

  1. Open Terminal
  2. sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

Config

Open zsh config file: nano ~/.zshrc.

Some suggesting plugins:

  • zsh-syntax-highlighting
  • zsh-autosuggestions
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions

After install the plugins above, change ~/.zshrc plugin setting:

plugins=(git z zsh-autosuggestions zsh-syntax-highlighting)

And add these lines at the bottom of file to display current time in prompt input:

export PATH="/usr/local/opt/gettext/bin:$PATH"
PROMPT='%{$fg[yellow]%}[%D{%Y-%m-%d} %D{%L:%M:%S}] '$PROMPT

Finally, source ~/.zshrc to make modification effective.
